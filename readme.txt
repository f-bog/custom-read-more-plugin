This plugin allows you to change the default length of your excerpt, and allows you to change the hover animation and text of the read more link. 

== Description ==
FWD Read More is an excerpt customizer plugin.
Simply use the WordPress admin to edit your excerpt text length, and change the wording of your read more link.
You have 3 options for hover animations: Spread, Up, and Pulse.

Options can be found in the WordPress dashboard menu under (FWD Read More) 

Through the options panel you can: 
* Customize read more text
* Customize excerpt length 
* Select a hover effect for the read more length

This is our first entry into plugin development for WordPress, if you notice any bugs or issues please let us know.
 

 
== Installation ==
 
This section describes how to install the plugin and get it working.

1. Upload `/fwd-readmore/` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the options located in the WordPress dashboard menu 
4. Customize!
